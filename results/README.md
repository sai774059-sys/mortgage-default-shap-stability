# Results

This folder stores output files generated by the notebook during the SHAP analysis and stability testing phases. Files are saved automatically when the corresponding notebook cells are run.

## Contents

After running the full notebook, this folder will contain the following outputs:

```
results/
|-- shap_values_pre_covid.npy          # SHAP values for pre-COVID sample (10,000 obs)
|-- shap_values_post_covid.npy         # SHAP values for post-COVID sample (10,000 obs)
|-- shap_pre_ranking.csv               # Mean absolute SHAP rankings, pre-COVID period
|-- shap_post_ranking.csv              # Mean absolute SHAP rankings, post-COVID period
|-- bootstrap_kendall_w.csv            # Kendall's W scores across 30 bootstrap runs
|-- perturbation_results.csv           # Spearman correlations at 0%, 5%, 10%, 15% noise
|-- smote_shap_ranking.csv             # SHAP rankings from SMOTE-trained model
|-- classweight_shap_ranking.csv       # SHAP rankings from class-weighted model
```
